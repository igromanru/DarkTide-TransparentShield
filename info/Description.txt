[font=Verdana][size=4][b]Description[/b][/size][/font]
Simple mod that makes Ogryn's Slab Shield transparent. You can choose the opacity in mod's options.
[size=2]Good for people with claustrophobia.[/size]

[font=Verdana][size=4][b]Requirements[/b][/size][/font]
[size=3]- [url=https://www.nexusmods.com/warhammer40kdarktide/mods/19]Darktide Mod Loader[/url]
- [url=https://www.nexusmods.com/warhammer40kdarktide/mods/8]Darktide Mod Framework[/url][/size]

[font=Verdana][size=4][b]Installation[/b][/size][/font]
1. Read descriptions and install [url=https://www.nexusmods.com/warhammer40kdarktide/mods/19]Darktide Mod Loader[/url] and [url=https://www.nexusmods.com/warhammer40kdarktide/mods/8]Darktide Mod Framework[/url]
2. In [b]Files[/b] tab use the [b]Mod Manager Download[/b] button to install the mod with [url=https://www.nexusmods.com/about/vortex/]Vortex[/url].
3. Make sure the mod is enabled in Vortex's [b]Load Order[/b] tab.
4. Done.

[font=Verdana][size=4][b]Report Bugs[/b][/size][/font]
[size=2]After changing mod's option, the changes should take effect at the latest after switching to another weapon and back to the shield.[/size]
If you spot any problems or inconsistency with the mod.
Please write a comment or create a bug report and try to describe as well as possible how the problem can be reproduced.
